# Prim'O — Admin API test report

_Generated by `tests/admin/run.sh` against `http://localhost:4000/api`. Tokens shown as `$TOKEN`._

## 1. Auth setup

Login as admin to obtain the bearer token used below:

```bash
curl -X POST "http://localhost:4000/api/auth/login" \
  -H 'Content-Type: application/json' \
  -d '{"email":"admin@primo.fr","password":"password123"}'
# => { "accessToken": "<jwt>" }   # export TOKEN=<jwt>
```

## 3. Users — auth & validation guards

```bash

```

**→ HTTP 401**

```json

  "error": "Token manquant."
}
```

```bash
curl -X GET "http://localhost:4000/api/admin/users" \
  -H "Authorization: Bearer $TOKEN"
```

**→ HTTP 403**

```json
{
  "error": "Accès réservé aux administrateurs."
}
```
